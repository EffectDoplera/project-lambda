# OMDB

Нужно реализовать приложение на указанном стеке технологий, которое имеет 2 страницы и у первой есть 2 состояния.
В начале, когда мы заходим на страницу, то видим пустую страницу со строкой поиска, введя название фильма мы отобразим пришедшие результаты в виде карточек фильмов с кратким описанием. Также мы можем кликнуть на эту карточку и перейти как раз на вторую страницу, где будет подробное описание этого фильма. Оттуда мы можем вернуться назад. Также должны быть реализованы фильтры, сортировки. И после этого также нужно будет написать тесты.
